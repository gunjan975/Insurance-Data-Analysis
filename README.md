# Insurance-Data-Analysis
# 📊 Insurance Data Analysis Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard analyzing insurance data to uncover key insights related to premiums, claims, policy types, and customer behavior. The dashboard helps in understanding patterns that influence insurance costs and claim trends.

---

## 🎯 Objectives

* Analyze insurance data to identify key cost drivers
* Understand claim distribution and policy performance
* Visualize customer demographics and behavior
* Provide actionable insights for better decision-making

---

## 🛠 Tools & Technologies

* **Power BI Desktop**
* Data Cleaning & Transformation
* Data Visualization
* Basic Data Modeling

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Premium Amount: **5.98M**
* Total Coverage Amount: **600.55M**
* Total Claim Amount: **16.91M**

### 🔹 Visual Insights

* Premium distribution by policy type
* Active vs inactive policies
* Claim count by status (Rejected, Settled, Pending)
* Claim amount analysis by age group
* Policy-wise claim breakdown

### 🔹 Customer Insights

* Gender-based distribution
* Age group segmentation (Adult, Elder, Young Adult)

### 🔹 Feedback Analysis

* Word cloud visualization highlighting customer feedback
* Identifies common themes like service quality, claims process, and satisfaction
